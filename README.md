
# Anime Next Watch Generator

A site where you can search and find anime to watch next.

---

## About

I made this as my semester project. You can search for anime, filter by genre, sort by rating, and save your favorites. Data comes from a free API.

---

## API used

Jikan API - https://jikan.moe
No API key needed.

---

## Features

- Search by title
- Filter by genre
- Sort by rating
- Save favorites
- Dark/light mode

---

## Built with

HTML, CSS, JavaScript

---

## How to run

Download or clone the repo, then just open index.html in your browser.
